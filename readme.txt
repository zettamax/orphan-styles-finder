Реализация задания для номинации 1+

Скрипт умеет:

1) Получать список неиспользуемых стилей для сайта, можно указать кол-во страниц для обработки (по умолчанию 5)

2) Получать список неиспользуемых стилей для одной или более указанных страниц

Из сторониих библиотек используется tinycss для парсинга css, благодаря этому есть полная поддержка правил @import и @media.
Вопрос с псевдоклассами / псевдоэлементами решил следующим образом - удаляю их из селектора, и если получившийся селектор подходит, то считаю что исходный тоже подходит
На внешний вид упор не делал

Для запуска я использовал Apache с модулем wsgi (как наиболее простой для меня способ)
Файл wsgi.py используется как точка входа
